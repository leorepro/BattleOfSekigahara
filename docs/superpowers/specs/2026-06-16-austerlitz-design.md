## 1. 敘事主軸（已確認）

**A. 「拿破崙的陷阱」全局謀略視角。**

全片以拿破崙的誘敵詭計為骨幹，不是孤立的會戰寫實，而是「為什麼這場是經典」的謀略展演。主線時間軸：

1. **設餌**：法軍主動放棄戰場中央戰略要地**普拉欽高地**，刻意暴露虛弱的右翼（南線塔爾尼茲／索科爾尼茲僅一個師）。
2. **咬餌**：聯軍（亞歷山大一世主導，否決庫圖佐夫的持重之議）將主力四個縱隊南調，圍攻法軍右翼，企圖切斷法軍退維也納之路並向北包抄。
3. **春雷**：「奧斯特利茨的太陽」破雲散霧的瞬間（約上午 9 點），蘇爾特軍旺達姆師＋聖海拉爾師（潛伏高地腳下的 16,000 人）敲鼓挺刺刀一舉衝上**兵力空虛的普拉欽高地**——中央突破。
4. **斬腰**：法軍佔領高地、拿破崙移大本營上高地，貝爾納多特自北線增援，俄國近衛軍孤注一擲反攻被擊退。聯軍被**切成兩半**。
5. **冰湖**：聖海拉爾師南下抄截，南線聯軍歸路被斷，潰兵湧向冰封的扎錢湖，高地火砲下轟冰面。
6. **終局**：第三次反法同盟瓦解、神聖羅馬帝國次年解體。拿破崙成歐洲霸主。

運鏡與字幕全程扣住「誘敵→咬餌→突破→斬腰→潰敗」這條線，讓觀眾**看懂陷阱**。

**新增引擎模組**（拿破崙時代專屬）：

- `src/engine/napoleonic.js` — 拿破崙時代單兵幾何（步兵/騎兵/砲兵三大類 + 軍服頂點色 + LOD），對應溫泉關的 `hoplite.js`。
- `src/engine/maneuver.js` — **路徑關鍵影格移動系統**（approach ①），單位依時間軸 t 內插位置/朝向，含 `charge` 衝鋒段、空心方陣狀態切換。本戰核心新工程。
- `src/engine/volley.js` — 排槍齊射 + 野戰砲兵齊射 + 砲煙/彈著塵爆/槍口火光（擴充或複用 `effects.js` 的粒子/ShaderMaterial 路徑）。

> 模組劃分原則：每個新模組單一職責、對外以資料驅動介面溝通、可獨立測試。`maneuver.js` 只管「t → 單位 transform」；`volley.js` 只管「開火事件 → FX」；`napoleonic.js` 只管「variant → 幾何」。

## 3. 部隊移動系統（approach ①，已確認）

**路徑關鍵影格驅動 + 單兵微抖動（③的一點）。移動是時間軸 t 的純函數 → 完全可拖曳/倒退重現（專案硬需求）。**

- 每個單位在 `data/austerlitz/armies.js` 定義 `path: [{t, x, z, facing?, mode?}...]`：
  - `mode`：`march`（行軍縱隊）/ `line`（展開橫隊）/ `charge`（騎兵衝鋒，高速段）/ `square`（步兵結方陣）/ `hold`（定點接戰）/ `rout`（潰逃）。
  - `maneuver.js` 依 t 在相鄰 waypoint 間內插出單位中心位置與朝向；`charge` 段用 ease-in 加速曲線並觸發衝擊 FX；`square` 段切換 `formation.js` 的陣型為空心方陣。
- 單兵位置 = 單位中心 + 陣型內本地偏移（沿用 `formation.js`），再加**微抖動**（小幅 noise，由 t 決定 → 仍可重現）增加生氣。
- 兵貼地形（沿用溫泉關 `heightAt` 逐兵貼地 + clamp 高度差，不上山不落水）。
- 衝突收斂沿用既有 decollide/hitbox。

## 4. 新戰術機制（四項全做，分 Phase）

優先序與對應招牌畫面：

1. **騎兵衝鋒機動**（必做核心）：成隊騎兵（繆拉預備騎兵軍、列支敦斯登奧地利騎兵、俄國近衛騎兵）從陣列 ease-in 加速衝刺、撞擊敵線、追擊潰兵。`charge` path 段 + 衝擊瞬間塵爆/火花 FX + 子彈時間焦點（沿用 `cinemaScale`）。
2. **線列步兵 vs 縱隊 + 排槍齊射**：法軍 `march`→`line` 展開、聯軍橫隊；`volley.js` 一排排槍口火光（沿時間錯落，非全齊）+ 硝煙幕。招牌：法軍縱隊推進、橫隊排槍對轟。
3. **野戰砲兵**：成排火砲（`napoleonic.js` 砲車幾何）齊射、砲煙環、彈著塵爆。高潮：**普拉欽高地火砲砲擊冰湖**。
4. **步兵空心方陣抗騎兵**：受騎兵威脅的步兵單位在 path 標 `square` 段 → 結成空心方陣（`formation.js` 新 `formationStyle:'square'`），騎兵衝鋒繞陣或被擊退。

## 6. 地形與天氣

- **真實 DEM**：`fetch_dem_austerlitz.py`，bbox 約 **lon 16.68–16.88 / lat 49.06–49.20**（涵蓋普拉欽高地、桑頓山、Goldbach 溪、Telnitz/Sokolnitz、扎錢湖南緣）。中心 ~49.13N 16.76E。產 `data/austerlitz/heightmap.js`。**使用者自行跑真實 DEM**（沿溫泉關慣例）。
